# music-brainstorming-bgm

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Instrumental background music for brainstorming sessions.

## Features

- Uplifting, futuristic electronic grooves
- Light, minimalist melodies and synth arpeggios
- Steady, mid-tempo beats to keep energy flowing
- No distracting vocals
- Perfect for creative thinking and innovation

## Usage

To download the playlist:

```sh
deno -A https://code4fukui.github.io/music-opendata-fukui/download.js ed50f5df-5017-4e51-b88a-9aadf3ca025d
```

The playlist can also be played in the browser at https://suno.com/playlist/ed50f5df-5017-4e51-b88a-9aadf3ca025d

## Data / API

This project uses open audio data hosted by the [code4fukui/music-opendata-fukui](https://github.com/code4fukui/music-opendata-fukui) project.
