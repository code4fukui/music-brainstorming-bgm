# music-brainstorming-bgm

ブレインストーミング用のインスト背景音楽です。軽快な電子サウンドが創造性を刺激します。

## 機能

- 前向きで軽快な電子グルーブ
- 優しいシンセアルペジオとメロディ
- 思考を邪魔しないミニマルな構成
- 歌詞のない楽曲

## 使い方

以下のコマンドでプレイリストをダウンロードできます:

```sh
deno -A https://code4fukui.github.io/music-opendata-fukui/download.js ed50f5df-5017-4e51-b88a-9aadf3ca025d
```

プレイリストはまた https://suno.com/playlist/ed50f5df-5017-4e51-b88a-9aadf3ca025d でブラウザ上で再生することもできます。

## データ・API

このプロジェクトは、[code4fukui/music-opendata-fukui](https://github.com/code4fukui/music-opendata-fukui) プロジェクトが提供するオープンオーディオデータを使用しています。
